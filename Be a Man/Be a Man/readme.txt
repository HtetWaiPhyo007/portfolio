🔥 Be a Man - パーソナルフィットネスアプリ
=======================================

This app helps users create and follow a weekly personalized fitness plan 
based on their name, age, sex, height, weight, and preferred workout days.
各ユーザーが自分専用のアカウントを作成し、データは自動的に保存されます。

✅ Features / 機能一覧
---------------------
- 🔐 ログイン & サインアップ（パスワードなし、ユーザー名のみ）
- 🧑 ユーザー情報入力（名前、年齢、性別、身長［cm/inch］、体重［kg/lb］）
- 🧠 自動でBMI、目標（太る・痩せる・維持）、1日カロリー、水分摂取量、睡眠時間を計算
- 📅 曜日を選んで、トレーニング曜日を決定
- 🏋️ 賢い運動プラン（毎日4種類のエクササイズを自動で選出）
- ✅ チェックボックスで進捗を記録できる
- 💾 ユーザーデータを自動保存（`data/users.json`）

📂 Project Structure / プロジェクト構成
---------------------------------------
.
├── main.py             ← メインアプリコード（このファイル）
├── data/
│   └── users.json      ← 保存されるユーザーデータ（自動作成）

💻 Requirements / 必要環境
--------------------------
- Python 3.x
- Kivy（インストール方法： `pip install kivy`）

▶️ How to Run / 起動方法
-------------------------
1. `main.py`として保存してください。
2. ターミナルまたはコマンドプロンプトで以下を実行：
> python main.py

📌 Notes / 補足
----------------
- データは `data/users.json` に保存されます。
- ユーザー名を入力するだけでログイン可能。
- 存在しないユーザー名を入力すると、自動で新規作成されます。
- 各ユーザーのプラン・進捗は完全に分離されています。

これは学校で自分で初めて作った Website です。初めて作ったWebサイトなので、あまり完璧じゃない部分もありますが、一生懸命に取り組んだ大切な作品です。

中では JavaScript や CSS を使って動きをつけています。特に、画像にpointer を乗せると 映像が動きます。後はBrightness設定と翻訳を入れています。

ファイル構成：
- index.html：トップページ
- History.html, Culture.html, Geography.html, Travel.html：各カテゴリーページ
- Css/：デザイン用CSSファイル
- Js/：JavaScript（動きの部分）
- Photo/：写真など

ブラウザで index.html を開けばサイトが表示されます。
